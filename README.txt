=====================================================================
================================ README =============================
=====================================================================

PyF is a Python library for writing interactive fiction. It aims to
make IF authoring simple and efficient without a need to learn a
specialized language.

Project homepage: http://pyf.sourceforge.net/

=====================================================================
============================ INSTALLATION ===========================
=====================================================================

NOTE: 	Development versions of PyF might include setup scripts that
		are out of date. Installing a development version with the
		method specified below might cause problems.
		
		You can run development versions without installation by 
		copying PyF source folder (the one that contains __init__.py) 
		to your game folder.

To install PyF run this in command prompt:
python setup.py install
